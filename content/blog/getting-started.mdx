# Getting Started with Solana

Learn how to set up your environment for Solana development.

## Prerequisites

- Node.js 18+
- Rust installed
- Solana CLI tools

## Installation Steps

1. Install the Solana CLI:

```bash
sh -c "$(curl -sSfL https://release.solana.com/stable/install)"
```

2. Verify installation:

```bash
solana --version
```

3. Set up your local test validator:

```bash
solana-test-validator
```

Ready to start building!
